# Flash Anzan Web Verification Harness

This directory is the pre-implementation verification harness.

## Commands

```bash
npm install
npm run test:contracts
npm run test:unit
npm run test:sim
npm run test:e2e
npm run test:all
npm run audit:generator
npm run verify:ready
npm run verify:full
npm run smoke:capture
npm run smoke:screens
npm run smoke:screens:mobile
```

## Scope

- Profile contract validation
- Recommendation policy validation
- State machine validation
- Browser smoke validation
- Generator variety audit
- UI smoke capture
- Self-hosted preview smoke verification
- Full verification entrypoint (`verify:full`)

This is not the full app yet.
